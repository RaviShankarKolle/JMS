Movie Website

A React-based movie browsing app powered by the TMDB API.
Users can search movies, sort results, load more items with pagination, switch dark/light mode, and view full movie details in a modal.

## Features

- Search movies by title
- Sort movies by:
  - Year Ascending
  - Year Descending
  - Popular Movie List (rating-based in UI)
- Load more movies with pagination
- Movie details popup/modal
- Dark mode / Light mode toggle
- Error and loading states
- Unit tests with Jest + React Testing Library

## Getting Started

### 1) Clone the repository

```bash
git clone <your-repo-url>
cd movie-website
```

### 2) Install dependencies

```bash
npm install
```

### 3) Add environment variables

Create a `.env` file in the root folder and add:

```env
VITE_TMDB_API_KEY=your_tmdb_api_key_here
```

You can get an API key from [TMDB](https://www.themoviedb.org/settings/api).

### 4) Run the app locally

```bash
npm run dev
```

Then open the local URL shown in terminal (usually `http://localhost:5173`).

## How It Works (High-Level Flow)

1. User types in search bar (`Header`)
2. `searchTerm` updates in `useMovies` hook
3. Hook triggers fetch via `tmdbService`
4. API response is mapped to app-friendly movie model
5. `displayMovies` is rendered in `HomePage` / `SearchPage`
6. Clicking a movie opens modal with full details
